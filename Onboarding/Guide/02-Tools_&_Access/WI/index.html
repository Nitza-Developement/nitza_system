    Guía de Bienvenida - Desarrollo de Software

    2- Herramientas y Accesos

    2.1 Sobre Nosotros.

    Al unirte al equipo, recibirás por WhatsApp los enlaces de acceso para:

    • Zulip (comunicación interna)
    • Odoo (gestión de proyectos y reporte de horas)
    • Repositorios correspondientes (GitHub/Azure DevOps)
    2.2 Plataformas de Comunicación.
    • Zulip Comunicación principal del equipo
    • Microsoft Teams Comunicación con clientes específicos
    • Google Meet Reuniones internas bi-semanales
    2.3 Plataformas de Comunicación.
    • Odoo:
      • Gestión de proyectos
      • Registro de horas trabajadas
      • Seguimiento de tareas
    2.4 Plataformas de Comunicación.
    • Repositorios distribuidos entre GitHub y Azure DevOps
    • Proceso de Pull Requests supervisado por desarrolladores senior designados
    2.5 Accesos Especiales.
    • ZeroTrust: Configuración requerida para acceder a servidores de clientes específicos
      • La configuración se proporcionará cuando sea necesario
